General Settings

Dark Mode

 Dark mode shifts the color pallate into a less bright white and a darker easier on the eyes charcoal and black (Off by default)

NCD Tool Settings

Supress Duplicates

 This setting effects wether the tool automatically supresses duplicate procedure and diagnosis codes, if on, no duplicate codes will be processed. If set to off, all codes put into the tool will be processed. (On by default)
